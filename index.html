UI Components

Our component library provides consistent, reusable UI elements for building FC Shining Stars digital experiences.

Buttons

Buttons allow users to take actions and make choices with a single tap. Different styles communicate different types of actions.

Primary Button
Used for primary actions and main calls-to-action.
Secondary Button
Used for secondary actions that complement the primary action.
Outline Button
Used for less prominent actions that still require emphasis.
Text Button
Used for the least prominent actions, often in cards or dialogs.
Disabled Button
Indicates that a button action is unavailable.

Match Schedule with Tabs
Interactive tabbed interface for displaying upcoming matches and past results.

Match Schedule JavaScript

// Schedule Tabs
const scheduleTabs = document.querySelectorAll('.schedule-tab');
const schedulePanels = document.querySelectorAll('.schedule-panel');

scheduleTabs.forEach(tab => {
    tab.addEventListener('click', () => {
        // Remove active class from all tabs
        scheduleTabs.forEach(t => t.classList.remove('active'));

        // Add active class to clicked tab
        tab.classList.add('active');

        // Hide all panels
        schedulePanels.forEach(panel => panel.classList.remove('active'));

        // Show corresponding panel
        const panelId = tab.getAttribute('data-tab');
        document.getElementById(`${panelId}-matches`).classList.add('active');
    });
});
